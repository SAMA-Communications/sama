SAMA features

Powerful but easy in use alternative to existing real-time messaging platforms and services.

Messaging

State of the art messaging API, with rich media messages, statuses, with a way to send to a user, a group of users

Private and group chats
Typing/Sent/Read statuses
File attachments
Messages history API

Users management

Powerful users management with authorization, edit profile, users search and block list supported.

User authorization
Edit profile
Users search
Block list

Multi devices support

Server allows for people to be always connected using multiple devices and flexible Online/Offline mesages delivery

Multiple user devices supported
Online/Offline mesages delivery
Deliver message to all user's devices

Last activity

Server keeps track who is online, offline and provides a handy API to manage it in client apps

Manage online, offline status
Retrieve user's last activity
Last activity subscription API

Push notifications

Receive messages even if offline via alternative message delivery methods - Push Notifications

Receive messages even if offline
APNS/FCM/Web push notifications supported

Clustering

Server clustering supported out of the box to support large loads / large user base over extended period of time

Support large loads
Support large user base

Case Studies

Take a quick tour through the set of the projects that we have successfully delivered to out clients

Team messenger & video calling collaboration app

The unified communications messenger and video calling app with 1-1 and group messaging, group video calling, contacts list, offline mode and push notifications.

All the modern messaging & video callings apps capabilities are supported. The app is built for Mobile, Web and Desktop which allows any platform communication from any place in the world.

Sharing economy chat bot

Share your spare food, books, sport equipment or whatever you share in your daily life using Sharing economy chat bot. The bot is able to make our world a little bit greener and will help humans stay open and connected to each other.

The bot is built for Facebook Messenger, including natural language processing capabilities, cloud storage for shared items, location discovery, wide shared items discovery capabilities.
